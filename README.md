# Deno Demo

A very simple project for a demo.

## DB

Navigate to deno-demo-db folder:

```cmd
docker-compose up -d
```

## APP

Navigate to deno-app folder:

```cmd
deno run --allow-net --allow-write --allow-read --allow-plugin --allow-env --unstable server.ts
```
